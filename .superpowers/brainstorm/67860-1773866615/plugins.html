Plugin & MCP Management

Manage everything from Slack — Claude plugins, MCPs, and SlackClaw extensions

Commands
@SlackClaw plugins — list all installed plugins, MCPs, and bot extensions
@SlackClaw add plugin superpowers — install Claude Code plugin
@SlackClaw remove plugin superpowers — uninstall Claude Code plugin
@SlackClaw add mcp github npx @modelcontextprotocol/server-github
@SlackClaw remove mcp github
@SlackClaw add bot-plugin app-store-connect — SlackClaw extension

How it works under the hood

Claude plugins → shells out to claude plugin install/uninstall on the host machine
MCPs → shells out to claude mcp add/remove, updates ~/.claude/settings.json
SlackClaw extensions → downloaded to SlackClaw/extensions/, loaded at runtime, no restart needed

All changes take effect on the next run: session. Hot-reload for bot extensions.
